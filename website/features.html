Every Feature You Need

MailVault is an email client and backup tool for macOS and Linux. From unified inbox and threaded conversations to local archiving, here's everything it can do.

Email Management Threading Chat View Security Storage Bulk Operations Multi-Account Customization Performance Premium

Email Management

Everything you need to read, compose, and organize your email. Sign in with one click, search with precision, and manage your inbox effortlessly.

  • Compose, reply, reply all, and forward with attachments
  • Gmail & Microsoft 365 OAuth2 one-click sign-in
  • Microsoft Graph API for Outlook.com — bypasses IMAP regression
  • Advanced search with filters — sender, date range, attachments, folder
  • Search history & popular filter suggestions
  • Mark read/unread, star/flag emails
  • Undo send — configurable delay to cancel outgoing emails
  • Move to folder — reorganize emails between server folders
  • Email templates — save and reuse compose snippets
  • Group by sender — toggle to organize your inbox by contact, with sent replies threaded inline
  • Link safety — detect phishing links and sender impersonation with visual warnings

Email Threading & Conversations

Gmail-style threaded conversations powered by the JWZ algorithm. See related emails grouped together, with smart folding for a clutter-free reading experience.

  • Gmail-style threaded conversations using the JWZ algorithm
  • Thread conversation view — all emails stacked chronologically
  • Configurable sort order — oldest-first or newest-first
  • Quote folding — quoted replies collapsed for cleaner reading
  • Signature folding — 4 modes: Smart, Always Show, Always Hide, Collapsed

Chat View

A unique messaging-style email experience. Sent and received emails merge into one continuous conversation with chat bubbles, just like your favorite messaging app.

  • Unique messaging-style email view
  • Sent and received emails merged into one conversation
  • Chat bubbles with sender avatars
  • Progressive body loading with per-bubble spinners

Security & Privacy

Your emails stay on your machine. MailVault verifies senders, stores credentials in macOS Keychain or Linux Secret Service, and runs sandboxed. No cloud services, no telemetry.

  • Link safety — detects phishing links where displayed text doesn't match the actual URL destination
  • Sender impersonation detection — warns when display name fakes a different email or domain
  • Click confirmation — modal warning before opening suspicious links with URL comparison
  • Sender verification badges — SPF, DKIM, and DMARC checks
  • Header mismatch detection — From vs Reply-To spoofing alerts
  • macOS Keychain & Linux Secret Service credential storage
  • macOS App Sandbox — runs in a secure container
  • No cloud, no tracking, no telemetry
  • Open source — inspect every line of code

Storage & Backup

Your emails stored in standard formats you can read anywhere. Archive locally with one click, export as ZIP, and restore on any machine. No vendor lock-in, ever.

  • Standard Maildir + .eml (RFC 5322) format — never locked in
  • One-click archive to local storage
  • ZIP backup export — archived only or all emails, with settings
  • Portable import — restore on any machine
  • Configurable cache size limit and duration
  • Background caching pipeline — syncs silently

Bulk Operations

Select hundreds or thousands of emails at once and archive, delete, or do both in a single operation. Pick a date range, watch the progress, and cancel anytime.

  • Multi-select with floating action bar
  • Batch archive, delete, or archive + delete
  • Date range selection — by year or custom range
  • Progress bar with cancel button
  • Crash-safe operation recovery

Multi-Account

All your email accounts in one place. Switch between them instantly — no loading delay. Combine them into a unified inbox, or customize each with its own display name and color.

  • Add unlimited email accounts
  • Unified inbox — see all accounts in one merged view
  • Gmail, Outlook, Yahoo, iCloud, ProtonMail (Bridge), Fastmail, Zoho, and any IMAP provider
  • Custom display names and avatar colors
  • Account reordering and hide/unhide
  • Instant account switching with LRU state cache
  • Per-account mailbox memory — remembers your last folder
  • Sender insights — exchange history and frequency per contact

Performance & Technology

Built with Rust and Tauri for native speed and minimal resource usage. Smart sync protocols, virtual scrolling, and pre-fetching keep everything fast.

  • Built with Rust (Tauri) — not Electron
  • Native IMAP via async-imap — no Node.js sidecar
  • CONDSTORE delta sync — zero IMAP calls when nothing changed
  • COMPRESS=DEFLATE — 70-80% bandwidth reduction
  • Virtual scrolling — handles 17,000+ emails smoothly
  • Email body pre-fetching — next 3 emails loaded instantly
  • Automatic updates via built-in updater
  • System tray — syncs silently in background
